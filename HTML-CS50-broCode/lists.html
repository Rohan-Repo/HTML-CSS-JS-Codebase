Technology Overview

Front-End Technologies
Languages which showcase the Look-And-Feel of a Website or App like HTML, Bootsrap or ReactJS
Back-End Technologies
Languages which run in the background usually connected to another system or Database like Java or Python
Databases
Where the Data gets Stored either in Relational (Rows/Columns) Format like SQL-Server or MySQL or in Documents or Graphs like MongoDB or Neo4J
Mobile-Apps
Apps Created for Android or iOS devices using languages like Kotlin or React-Native or Swift
